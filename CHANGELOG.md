## [0.2.0] - 2025-06-10

### Added

- **Exercise Library**: Introduced an interactive Body Map Explorer for muscle-based exercise filtering.
- **Workout Enhancements**: Added RPE (Rate of Perceived Exertion) tracking and intelligent weight suggestions based on historical data.
- **Sleep Tracking**: Added Google Fit synchronization for automated sleep data logging.
- **Progress Photos**: Migrated to Cloudinary for secure and efficient photo management.
- **Tech Stack Upgrade**: Upgraded project to React 19 and Material UI v7+.

### Changed

- Updated project documentation to reflect new features and architectural changes.
